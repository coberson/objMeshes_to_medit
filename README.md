Translating obj-meshes for use with mesh visualization software medit (Prof. Pascal Frey https://www.ljll.math.upmc.fr/frey/software.html).

C-function translate_mesh_for_medit.c

example: ./a.out spot/spot_triangulated.obj

The public domain blub, spot and Csaszar obj-meshes are taken from the repository 
http://www.cs.cmu.edu/~kmcrane/Projects/ModelRepository/ (Keenan Crane).
